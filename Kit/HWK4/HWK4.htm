Assignment

Write a Decrypt program that helps decrypt substitution ciphers.


  1. Accept the files to be decrypted as commandline arguments. Open each and decrypt it as described below. Be sure to close each one.

  2. For each file, first do an initial attempt at automatic decryption. Assume that the frequency of letters in English is, in descending order: etaoinshrdlcumwfgypbvkjxqz Count how often each letter occurs in the encrypted file, and match the most common one with e, the next most common one with t, etc. If two letters are equally common in the encrypted file, assume the one higher in alphabetic order is the “more frequent”. With the mapping thus derived, perform and display a crude decryption of the file, for the user to improve upon. Display “Enter command:” and await the user's input.

  3. The user may now enter lines comprising one or more space-separated “map correction” pairs, each in the form x->y. Such a pair means “turn all x's, upper or lower case, to y's, and vice versa”. Apply each pair in succession, and then reprint the text and the “Enter command:” prompt.

  4. Once the user is satisfied with the decryption, he enters “None”. The program writes out the decrypted version, to the name <original file name>.dec.

  5. Total non-blank, non-comment lines must be at most 75, subject to the same style rules as HW 2. (Mine's 65, and I was kinda sloppy.)

  6. See the model input and output to disambiguate the spec.
